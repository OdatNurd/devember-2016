About A-Maze-Balls

This is a simple clone of Bolo Ball by Soleau Software, created as a part of my Devember 2016 project.

This is a not-entirely-faithful port of the original game. The mechanics are somewhat the same as the original, although I have no extensive experience of that version and so this is more an exercise in replicating something and then extending it.

To that end, once the core gameplay is completed, additional mechanics will be added to the game in order to see how that changes things. Maybe we'll end up with something better, or maybe we will not.

Some of the graphical assets used here are from (or based upon): https://kenney.itch.io/kenney-game-assets-2

A-Maze-Balls Controls

F
Toggle fullscreen mode on the canvas (if supported by the browser)
G
Regenerate the level
T
Toggle debug/editing mode
Spacebar (debug/editing mode only)
Toggle the debug brick between subtypes
